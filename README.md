# Dimlmgr (Hanterare för Diml-miljöer)

Dimlmgr är en tjänst som hanterar information om Diml-miljöer:

* Vilka dataprodukter som är installerade
* Operationell metadata för dataprodukterna
* Connectionsträngar och sådant för anslutning till dataprodukterna
* Diverse konfiguration

[Diml](https://github.com/Region-Skane-SDI/diml) är ett metadataspråk för att beskriva digitala tillgångar.

## Innehåll
| Mapp | Beskrivning |
| -------- | ----------- |
| dimlmgr-client | Kod för Dimlmgr klient |
| dimlmgr-cmdline | Kommandoradsverktyg för att arbeta med dimlmgr |
| dimlmgr-server | Kod för Dimlmgr server |
| dimlmgr-test | Kod för automatiserade tester |
